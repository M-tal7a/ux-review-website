# UX Review Website

A clean and responsive **UX Blog Layout** built using HTML and CSS.

This project demonstrates a simple blog interface that focuses on user experience topics, modern layout structure, and responsive design.

---

## Features

- Clean UI Design
- Responsive Layout
- Blog Article Cards
- Author Avatars
- Font Awesome Icons
- Modern Typography

## Technologies Used

- CSS3
- Font Awesome
- Responsive Design
